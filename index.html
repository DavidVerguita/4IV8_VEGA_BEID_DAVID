Objetivo del Portafolio

Compilar, estructurar y presentar las soluciones algorítmicas y sistemas web desarrollados durante el ciclo escolar. Este espacio funge como evidencia del dominio adquirido en arquitecturas Cliente-Servidor, manipulación dinámica del DOM mediante JavaScript Vanilla, diseño responsivo estandarizado y la persistencia de datos relacionales a través de APIs REST nativas.

Elementos Aprendidos en la Materia

  • Fundamentos de la Arquitectura Web: Comprensión profunda del modelo Cliente-Servidor y el ciclo de vida de las peticiones HTTP empleando métodos nativos (GET, POST, PUT, DELETE).
  • Estándares de Diseño y Estructura: Maquetación semántica con HTML5 y estilización avanzada con CSS3, aplicando conceptos de maquetación limpia a interfaces interactivas oscuras.
  • Lógica e Interactividad en el Cliente: Programación bajo el estándar ECMAScript 6 (ES6), validación estricta de formularios mediante expresiones regulares y manipulación avanzada del DOM.
  • Desarrollo Backend y Persistencia: Creación de servidores web nativos con Node.js (módulo http), enrutamiento dinámico de archivos estáticos y operaciones transaccionales completas (CRUD) en bases de datos MySQL.
